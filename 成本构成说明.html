# 主材费

   由算量后的材料费用核定的总费用

   一般按系统进行计算

# 人工费

    由算量后的人工相关费用进行计算后的总费用

    由于与人工核定的材料与系统相关，人工费也按系统核定

# 辅材费

    在成本测算时根据最后的预算总价按比例进行测算

# 预算费用

    是指在成本测算时会计入到工程总价中的费用项

    一般不会在按系统测算的过程中使用

# 实施费用

    是指在成本测算时用于控制系统成本的费用项

    不会计入到工程总价进行进行核算


费用的计算方式有三种：

1. 固定金额

    直接设定费用金额，一般由成本人员根据项目的情况进行设定

2. 按项目面积核算

    根据项目的面积进行核定，此时需要指定单位面积的单价

3. 按比例进行核算

    一般按系统根据总价进行核定
